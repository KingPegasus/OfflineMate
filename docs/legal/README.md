**Website**

- Public site: [offlinemate.com](https://offlinemate.com) (source: [`website/`](../../website/)).
- Visitor-facing summary: [Privacy](https://offlinemate.com/privacy.html); full policy text remains in this repo (`privacy-policy.md`) and should stay in sync when you change either copy.

**Publishing for Google Play / App Store**

- Host these files in the public repo and use the **raw or rendered GitHub URL** in the store “Privacy policy” field, **or** mirror the same text on [offlinemate.com](https://offlinemate.com).
- Replace **placeholder contact** lines in both documents before production.
- Keep store **Data safety** and listing copy **consistent** with the privacy policy.
